# Web Game Store

## Description
Web Game Store is an online platform for purchasing and downloading digital games. Built using JSP and Spring MVC, this project provides a seamless user experience and a robust backend for managing game listings, user accounts, and transactions.

## Features
- User Registration and Login
- Browse and Search Games
- Game Details Page
- Shopping Cart
- Secure Payment Integration
- User Profile Management
- Admin Dashboard for Game Management

## Technologies Used
- **Java**: Programming language for the backend
- **JSP**: JavaServer Pages for dynamic web content
- **Spring MVC**: Framework for building the web application
- **MySQL**: Database for storing user and game information
- **HTML/CSS/JavaScript**: Frontend technologies

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/dvhlong/WebGameStore_JSP_SpringMVC.git
   ```
2. Navigate to the project directory:
   ```bash
   cd WebGameStore_JSP_SpringMVC
   ```
3. Configure the database by importing the provided SQL file.
4. Set up the application server (e.g., Apache Tomcat) and deploy the project.
5. Access the application at `http://localhost:8080/WebGameStore`.

## Usage
- Register for a new account or log in to an existing one.
- Browse the available games and add them to your shopping cart.
- Proceed to checkout to purchase games.
